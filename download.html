Official Releases

To download and play Myst Online: Uru Live again, please use the installer from the Myst Online website.

To play on the Guild of Writers Gehn shard, please use the installer from the Guild of Writers website.

Development

Experimental development builds are available for download as artifacts from our GitHub pre-release tag. These are built automatically when code is merged, and are not tested for stability. These should be used for local testing only.

The game engine source code is also available for download from GitHub. For compiling instructions, please see the Development section.

For instructions on how to run these game clients, see the section on Running Plasma.

Game Assets

Additional game data files from Myst Online: Uru Live again are also available from the H-uru/moul-assets GitHub repository. These assets are licensed under the Creative Commons Attribution-NonCommercial-ShareAlike 4.0 (CC-BY-NC-SA 4.0) Licence.
